\section{Karten - Idee}
Die \emph{Karten}-App stellt die Datenpunkte geographisch auf einer Karte da. So können Nutzer Datensätze entdecken, die in ihrem eigenen Leben eine große Rolle spielen. Die Datensätze sollten mit Pins auf der Karte dargestellt werden, die geklickt werden können. 
\subsection{Karte}
Eine Karte macht es Nutzern einfach, Daten, die geographisch für Sie relevant, sind zu finden. So können Nutzer auch wenn sie unterwegs sind und etwas Interessantes entdecken sofort Informationen darüber finden.

\subsection{Downloads}
Nutzer sollen Datensätze, die sie besonders interessant finden oder auf die sie Online Zugriff haben können herunterladen können.

\subsection{Einstellungen}
In den Einstellungen kann die App für den Nutzer angepasst werden. So kann zwischen Hell- und Dunkelmodus gewechselt und persönliche Informationen angegeben werden.

\subsection{Ziel}
Ziel dieser App ist es, dass Nutzer leicht anhand geographischer Pins Informationen über ihre Stadt und ihre Nachbarschaft finden können.
