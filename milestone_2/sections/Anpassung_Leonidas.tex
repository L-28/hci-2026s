\section{Anpassung Leonidas}
Der Prototyp wurde für die in Meilenstein 1 definierten Personas angepasst.

\subsection{Benjamin (technikaffiner Student)}
Für Benjamin wurde die Suche verbessert, so können nun mehrere Filter zusammen funktionieren und reguläre Ausdrücke in der Suche verwendet werden. Dies ermöglicht es ihm, leichter interessante Datensätze zu finden.

\subsection{Leah (Content Creator)}
Für Leah wurde die Größe der Pins von der Popularität der zugrundeliegenden Datensätze abhängig gemacht. So kann sie schnell populäre und möglicherweise interessante Datensätze finden. Auch wurden mehr Features zur Sortierung hinzugefügt.

\subsection{Ralph (professioneller Nutzer)}
Für Ralph wurden auch Quellen und Nutzungslizenzen zu den Daten hinzugefügt und leicht filterbar gemacht. So können alle Daten automatisch ausgeblendet werden, die er aus Lizenzgründen nicht benutzen kann. Es können Pins jetzt auch nach Kategorie, Lizenz oder anderen Filtern farbcodiert werden, auch können Pins mit Filtern automatisch ausgeblendet werden. 


\subsection{Luca (Gelegenheitsnutzer)}
Für Luca wurde der Einstieg vereinfacht und logische Einstellungen als Standard genommen. So kann er die App leicht benutzen, ohne sie für sich anpassen zu müssen. So werden Pins automatisch farbcodiert.

\subsection{Problematischer Nutzer}
Für den problematischen Nutzer wurden keine Änderungen im Prototyp vorgenommen.
